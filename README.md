# Elara Project
A digital character that grows through community interactions and blockchain activity.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the portal: `python src/portal/app.py`
3. Generate mood images: `python src/plugins/elara_mood_plugin.py`
